## 📌 Project Overview
RetailPulse AI helps retailers:
- Forecast future sales & revenue
- Predict product demand
- Detect inventory risks
- Analyze customer behavior (RFM + CLV)
- Generate AI business insights
- Optimize revenue with ML

## 🛠 Tech Stack
| Layer | Technology |
|---|---|
| Frontend | Next.js, React, Tailwind CSS, Framer Motion, Recharts |
| Backend | FastAPI, Python |
| ML | Scikit-Learn, MLxtend, Pandas, NumPy |
| Database | PostgreSQL (planned) |

## 📁 Project Structure
```
RetailPulse-AI/
├── backend/          # FastAPI backend + ML routers
├── frontend/         # Next.js enterprise dashboard
├── src/              # ML modules (RFM, CLV, Forecasting, etc.)
└── data/             # Dataset (Online Retail)
```

## ⚡ Run Locally

**Backend:**
```bash
pip install -r requirements.txt
python -m uvicorn backend.main:app --port 8000
```

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:3000
